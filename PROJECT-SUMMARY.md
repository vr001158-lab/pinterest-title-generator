## 📞 Technical Support

### Common Issues

**Problem**: Tool doesn't load  
**Solution**: Check internet connection, clear browser cache, try different browser

**Problem**: Copy button doesn't work  
**Solution**: Use Ctrl+C / Cmd+C instead, or try different browser

**Problem**: Mobile layout looks weird  
**Solution**: Refresh page (Ctrl+Shift+R), close other tabs, try DevTools Device Toolbar

**Problem**: Search history disappeared  
**Solution**: It's device-specific. Clear browser data will wipe it. Normal behavior.
